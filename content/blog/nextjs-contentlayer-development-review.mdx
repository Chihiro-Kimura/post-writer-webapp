# Next.js + Contentlayer でブログ付き Web アプリケーションを開発した振り返り

## 📝 プロジェクト概要

Next.js 14 と Contentlayer を使用して、ブログ機能付きの Web アプリケーションを開発しました。主な機能として：

- ユーザー認証（GitHub/Google OAuth）
- ブログ記事の作成・編集・削除
- マークダウンによるコンテンツ管理
- ダッシュボード機能

を実装しています。

## 🛠 使用技術

### フロントエンド

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Editor.js

### バックエンド

- Prisma (ORM)
- PostgreSQL
- NextAuth.js
- Contentlayer

### インフラ

- Vercel (ホスティング)
- Supabase (データベース)

## 💡 工夫した点

1. **型安全性の確保**

   - TypeScript と Prisma の組み合わせによる堅牢な型システム
   - zod によるバリデーション
   - Contentlayer による型安全な MDX ハンドリング

2. **UI/UX**

   - shadcn/ui による一貫したデザインシステム
   - レスポンシブ対応
   - ダークモード対応

3. **認証システム**

   - NextAuth.js による堅牢な認証基盤
   - GitHub/Google OAuth の実装
   - ミドルウェアによる認証ルーティング制御

4. **パフォーマンス**
   - App Router によるサーバーコンポーネントの活用
   - Connection Pooling の実装
   - 画像の最適化
